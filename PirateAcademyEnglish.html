Pirate Academy



Pirate Academy is a WarioWare game realized by 44 Game Designer students in 2 months. The game uses the mechanics of crazy chains of mini-games of classic Wario Ware and add Rogue-like mechanics with procedural rewards and encounters.


Organisation:

Most of the students developed a single mini-game while few others worked and the global game. I was part of a team of three developers of the global game. Together we had to:
- Prepare a developer documentation for the mini-games.
- Plan the integration of the thirty mini-games.
- Develop the global game.


My role:

I created integrations tools to help the production of mini-games and their integration in the global game. I also created VFX and music integration tools so that the Sound Designers wouldn't interact much with Unity.

At mid-project I was asked to create QA tools so that QA team could improve the game and adjust it.
Here, the developers just had to reference their scene and their name and then press one button to have their game integrated in the global game.

Knowledge acquired:

- I learned to organise a project for 44 students with some that didn't know much about programming.
- I learned to create a clear and precise documentation.
- I perfected my knowledge of Tool Design on Unity by creating tools for fields that I never created one before like QA and Sound Design.
